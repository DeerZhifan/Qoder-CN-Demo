# 如何使用 CDP 租户管理功能

## 概述

租户管理实现多租户隔离，支持租户创建、禁用/启用、权限分配。创建租户时自动生成租户管理员账号（默认密码 `88888888`）。

## 核心接口

**Controller 路径前缀：** `/system/tenant`

| 接口 | 方法 | 说明 |
|------|------|------|
| `/system/tenant/page` | POST | 分页查询租户列表 |
| `/system/tenant/listOptions` | GET | 获取租户下拉选项 |
| `/system/tenant/add` | POST | 创建租户 |
| `/system/tenant/edit` | POST | 编辑租户信息 |
| `/system/tenant/disable` | POST | 批量禁用/启用租户 |
| `/system/tenant/validate/{code}` | GET | 校验租户编码唯一性 |
| `/system/tenant/clearCache` | GET | 清除租户缓存 |

## 使用流程

```
1. 使用超级管理员（superadmin/!QAZ2wsx）登录
2. 在【租户管理】中创建租户 → 自动生成租户管理员账号
3. 在【租户管理 - 分配权限】中给租户分配可用功能模块
4. 使用租户管理员账号登录（首次需修改密码）
5. 后续管理操作（用户、角色、组织等）在租户管理员账号下进行
```

## 注意事项

> 注意：创建租户后自动生成管理员账号，初始密码 `88888888`，首次登录强制修改。

> 注意：禁用租户后，该租户下所有用户均无法登录系统。

> 注意：`租户管理`、`资源管理`、`系统信息安全配置` 权限只应分配给超级管理员，不应分配给租户管理员。

> 注意：修改租户权限后建议调用 `/clearCache` 清除缓存，确保权限立即生效。
