# 如何使用 CDP 工作流 — 单据管理

## 概述

工作流单据管理负责业务单据与工作流程的关联，定义单据的审批页面 URL、表单变量映射等，使业务系统的数据能够在审批流程中正确展示和流转。

## 核心概念

- **Bill（单据）**：业务数据与流程实例的关联记录
- **BillCompany（单据公司）**：单据所属的公司/组织配置
- **Billurl（单据 URL）**：审批页面的访问地址配置
- **BillurlFormvar（表单变量）**：单据页面中表单字段与流程变量的映射

## 集成方式

业务系统需要：
1. 在工作流管理后台配置单据类型和对应的审批页面 URL
2. 配置表单变量映射（将业务字段映射为流程变量）
3. 在业务代码中调用流程发起接口，传入单据 ID

## 注意事项

> 注意：Billurl 配置的页面 URL 需要前端路由支持，审批人点击待办时会跳转到该页面。

> 注意：表单变量（BillurlFormvar）定义了流程中可用的业务数据，影响流程路由条件判断。
