# 如何使用 CDP 工作流 — 流程监控与报表

## 概述

工作流监控提供流程运行状态的实时监控和统计分析，帮助管理员了解流程效率、发现瓶颈并进行干预。

## 核心功能

- **流程监控（Monitor）**：查看所有流程实例的运行状态
- **运行报表（Report）**：流程执行的统计分析
- **首页看板（Home）**：流程概览数据
- **分析图表（Analysis/Chart）**：流程效率分析、环节耗时统计

## 管理员操作

- **暂停流程**：临时挂起正在运行的流程实例
- **恢复流程**：恢复被暂停的流程实例
- **终止流程**：强制结束流程（需管理员权限）
- **跳转环节**：将流程直接跳转到指定环节

## 注意事项

> 注意：流程监控和管理操作需要系统管理员权限（Sysmanage）。

> 注意：终止流程操作不可逆，终止后业务单据状态需手动处理。
